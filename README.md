# 🎬 Movie Search App

A simple web app that fetches movie data using the OMDb API. Built with HTML, CSS, and JavaScript.

## 🚀 Features

* Search movies by title
* Fetch real-time data from OMDb API
* Display movie details (title, year, poster, etc.)
* Clean and simple UI

## 🛠️ Tech Used

* HTML
* CSS
* JavaScript
* OMDb API

## ▶️ Live Demo

Check it out here: [Movie Searcher](https://daud-01.github.io/Movie-Searcher/)

## ▶️ How to Run Locally

1. Clone or download the project
2. Open `index.html` in your browser
3. Make sure your OMDb API key is added in the JS file

## 🎮 How to Use

* Enter a movie name in the search bar
* Click search
* View movie details instantly

## ⚠️ Note

* Requires internet connection
* Limited to OMDb API data
